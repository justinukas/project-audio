# project-audio
A simple CLI-based audio playback program written in C++ which uses the [miniaudio library](https://github.com/mackron/miniaudio)

- Features:
Loading and playback of audio files
Pausing
Stopping playback
Picking volume
Seeking
Checking elapsed time

Making of playlist files based on existing audio files in specified directory
Playback based on the playlist file
Skipping
